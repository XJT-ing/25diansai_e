# 2025 National Undergraduate Electronics Design Contest - Problem E

This repository contains the source code, schematics, 3D models, system block diagram, and project report for our 2025 National Undergraduate Electronics Design Contest Problem E project.

Award: National Second Prize

## Project Overview

This project implements a simple automatic aiming device based on machine vision, embedded control, motor actuation, and closed-loop correction.

## Repository Structure

- `源码/`: source code for MaixCam Pro and microcontroller control
- `原理图/`: circuit schematics
- `3D建模/`: 3D models and mechanical structure files
- `总体框图.png`: overall system block diagram
- `报告.pdf`: project report

## Note

This repository is intended for learning, communication, and project demonstration only.
